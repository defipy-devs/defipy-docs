## 1. Edit `conf.py` — re-enable nbsphinx

Open `docs/conf.py`.

**Edit A: Add `"nbsphinx"` back to the `extensions` list.** Place it after `sphinx.ext.autosummary`.

**Edit B: Remove `"**/*.ipynb"` from `exclude_patterns`.** Leave the other entries alone.

**Edit C: Restore the nbsphinx config block.** Add this at the bottom of the file, before any trailing comments:

```python
# -- nbsphinx configuration --
nbsphinx_prompt_width = "0"
nbsphinx_execute = "never"  # do not re-execute notebooks at build time
```

The `nbsphinx_execute = "never"` line is important — it tells nbsphinx to render notebooks using their stored cell outputs rather than re-executing them. Without this, RTD's build environment would need to successfully execute every notebook cell, which often fails for reasons unrelated to the docs (imports, file paths, etc.).

## 3. Fix nbconvert / jinja2 version pinning

The most common post-Python-upgrade nbsphinx break is nbconvert 7.x being pulled in when the existing toolchain expects nbconvert 6.x. Try in this order:

**Attempt 3A — pin nbconvert<7 and jinja2<3.1:**

Edit `docs/requirements.txt`. Add these lines in the hand-written (non-autogenerated) section near the top, after `nbsphinx`:

```
nbconvert<7
jinja2<3.1
```

Reinstall and rebuild:

```bash
pip install --upgrade --force-reinstall -r docs/requirements.txt
cd docs
sphinx-build -b html . _build/html 2>&1 | tee /tmp/build_nbsphinx_attempt3a.log
cd ..
```

If this succeeds — done, skip to step 5. If it fails, try 3B.

**Attempt 3B — upgrade nbsphinx instead.**

Revert the 3A pins and try upgrading nbsphinx itself to a version that handles nbconvert 7.x:

```
nbsphinx>=0.9.0
```

Reinstall, rebuild, check logs.

**Attempt 3C — if both fail.**

Pin both explicitly:

```
nbsphinx==0.9.3
nbconvert>=7.0,<7.17
```

0.9.3 is a recent nbsphinx release known to work with nbconvert 7.x. The upper pin on nbconvert avoids the 7.17+ template changes.

If none of 3A/3B/3C produce a clean build, stop and paste the failing build log excerpt into the summary.
